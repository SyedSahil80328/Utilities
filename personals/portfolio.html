Integrated Assistive System for Object Detection and Blind Navigation +

Empowering independence through technology: Explore our integrated assistive system combining advanced object detection and blind navigation, designed to enhance accessibility for individuals with visual impairments. Navigating the world with confidence and precision.

The system is developed using Raspberry Pi and is related to IoT. It uses Cascade Classifier for detecting Human Faces and SSD MobileNet for detecting and classifying 91 different objects. The system is designed to alert users by identifying objects using pyttsx3 (text-to-speech library) and OpenCV (computer vision library).

We got fund to implement this project from Centre for Sponsored Research and Consultancy, by Anna University through the scheme "Student Innovative Projects".
